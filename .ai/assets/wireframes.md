# 화면 설계 (와이어프레임)

## SCR-01. 홈 / 입력 화면

```
┌─────────────────────────────┐
│  Cluein                  ⚙️  │  ← 앱바 (설정 아이콘)
├─────────────────────────────┤
│                             │
│  ┌───────────────────────┐  │
│  │  "그거 있잖아~        │  │
│  │   생각나는 대로        │  │
│  │   입력해 보세요."      │  │  ← placeholder
│  │                       │  │
│  │                       │  │
│  └───────────────────────┘  │  ← 자유 입력 텍스트필드 (multiline)
│                             │
│  [불확실한 부분 마킹하기]     │  ← 확신도 마킹 버튼 (텍스트 선택 후 활성화)
│                             │
│        [기억 보정 시작]       │  ← 주요 CTA 버튼
│                             │
│  ─────── 최근 이력 ───────   │
│  • 사막에서 캡슐 기차 뉴스…  │  ← 최근 이력 미리보기 (최대 3건)
│  • 역할이 정해진 영화 2000…  │
│                             │
│        [이력 전체 보기]       │
└─────────────────────────────┘
```

**구성 요소**
- 앱바: 앱 이름 + 설정 아이콘
- 자유 입력 텍스트필드: 멀티라인, placeholder로 모호한 표현 유도
- 확신도 마킹 버튼: 텍스트 선택 시 활성화
- 기억 보정 시작 버튼: 입력 있을 때 활성화
- 최근 이력 미리보기 (SCR-06 축약)

## SCR-02. 확신도 마킹 화면

```
┌─────────────────────────────┐
│  ← 확신도 마킹           다음 │  ← 앱바
├─────────────────────────────┤
│                             │
│  긴가민가한 부분을           │
│  꾹 눌러 선택하세요.         │  ← 안내 문구
│                             │
│  ┌───────────────────────┐  │
│  │ 사막에서 캡슐 기차 타는│  │
│  │ 뉴스, [2005년]이었나? │  │  ← [ ] = 마킹된 구간 (흐릿한 배경)
│  └───────────────────────┘  │
│                             │
│  마킹된 항목:               │
│  • "2005년" — 긴가민가       │  ← 마킹 목록
│                             │
│  [마킹 완료 → 보정 시작]     │
└─────────────────────────────┘
```

**구성 요소**
- 입력 텍스트에서 불확실 구간 선택 (long-press or drag)
- 선택된 구간: 흐릿한 배경 또는 밑줄 태그 표시
- 마킹 목록: 선택된 불확실 항목 확인
- 보정 시작 버튼

## SCR-03. 추론 중 화면

```
┌─────────────────────────────┐
│  Cluein                     │
├─────────────────────────────┤
│                             │
│                             │
│         ◌ ◉ ◌              │  ← 로딩 애니메이션 (점 3개)
│                             │
│    기억을 분석하고 있습니다... │  ← 안내 문구
│                             │
│    단서를 조합 중입니다.      │  ← 서브 문구 (단계별 변경)
│                             │
│                             │
└─────────────────────────────┘
```

**구성 요소**
- 전체 화면 로딩 상태
- 로딩 인디케이터 (점 애니메이션 또는 원형 프로그레스)
- 단계 안내 문구: "기억을 분석하고 있습니다...", "단서를 조합 중입니다.", "거의 다 왔어요!"

## SCR-04. 보정 결과 화면

```
┌─────────────────────────────┐
│  ← 보정 결과             공유 │
├─────────────────────────────┤
│                             │
│  ┌── 입력하신 기억 ────────┐ │
│  │ 사막에서 캡슐 기차 타는  │ │  ← 원문 카드
│  │ 뉴스, 2005년이었나?      │ │
│  └─────────────────────────┘ │
│            ↓                │
│  ┌── 보정된 사실 ──────────┐ │
│  │ 2005년이 아닌 2016년    │ │  ← 보정 결과 카드
│  │ 네바다 사막 테스트 뉴스  │ │
│  │ 일 확률 95%             │ │
│  └─────────────────────────┘ │
│                             │
│  보정 이유:                  │
│  2005년에는 해당 기술이      │  ← 이유 설명
│  상용화 단계가 아니었습니다. │
│  Hyperloop의 첫 공개 테스트  │
│  는 2016년입니다.            │
│                             │
│       [출처 / 근거 보기]      │  ← SCR-05 이동
└─────────────────────────────┘
```

**구성 요소**
- 원문 카드: 사용자 입력 그대로 표시
- 보정 결과 카드: 수정된 사실 + 확률
- 보정 이유 설명 텍스트
- 출처/근거 보기 버튼 (SCR-05 이동)

---

## SCR-05. 출처 / 근거 화면

```
┌─────────────────────────────┐
│  ← 출처 / 근거              │
├─────────────────────────────┤
│                             │
│  ┌── 확인된 사실 ──────────┐ │
│  │ 📰 2016 Hyperloop 테스트 │ │  ← 뉴스 링크 카드
│  │    The Verge · 2016.05  │ │
│  │    [링크 열기 →]         │ │
│  └─────────────────────────┘ │
│                             │
│  ┌── 관련 영상 ────────────┐ │
│  │ ▶ Hyperloop One Test    │ │  ← 영상 카드
│  │   YouTube · 3:24        │ │
│  │   [재생 →]               │ │
│  └─────────────────────────┘ │
│                             │
│  ┌── 추측 항목 ────────────┐ │
│  │ ⚠️ "사막" 위치 — 확률 70% │ │  ← 출처 없는 추측 항목
│  │    (출처 없음, 추정)     │ │
│  └─────────────────────────┘ │
└─────────────────────────────┘
```

**구성 요소**
- 확인된 사실: 뉴스 링크 카드 (제목, 출처, 날짜)
- 관련 영상 카드 (선택적)
- 추측 항목: 출처 없이 확률만 표시, 경고 아이콘

## SCR-06. 이력 목록 화면

```
┌─────────────────────────────┐
│  ← 보정 이력                │
├─────────────────────────────┤
│                             │
│  2026-02-28                 │  ← 날짜 섹션 헤더
│  ┌─────────────────────────┐│
│  │ 사막에서 캡슐 기차 뉴스… ││  ← 이력 아이템
│  │ → 2016년 Hyperloop 테스트││
│  │               2026-02-28││
│  └─────────────────────────┘│
│                             │
│  2026-02-25                 │
│  ┌─────────────────────────┐│
│  │ 역할이 정해진 영화 2000… ││
│  │ → 가타카 (Gattaca, 1997) ││
│  │               2026-02-25││
│  └─────────────────────────┘│
│                             │
│  (스크롤하여 더 보기)        │
└─────────────────────────────┘
```

**구성 요소**
- 날짜별 섹션 헤더
- 이력 아이템 카드: 입력 원문 요약 + 보정 결과 요약 + 날짜
- 아이템 탭 → SCR-07 이동

---

## SCR-07. 이력 상세 화면

```
┌─────────────────────────────┐
│  ← 이력 상세            공유 │
├─────────────────────────────┤
│  2026-02-28                 │  ← 날짜
│                             │
│  ┌── 입력 원문 ────────────┐ │
│  │ 사막에서 캡슐 기차 타는  │ │
│  │ 뉴스, 2005년이었나?      │ │
│  └─────────────────────────┘ │
│                             │
│  ┌── 보정 결과 ────────────┐ │
│  │ 2016년 네바다 사막       │ │
│  │ Hyperloop 테스트 뉴스    │ │
│  │ (확률 95%)               │ │
│  └─────────────────────────┘ │
│                             │
│       [출처 / 근거 보기]      │  ← SCR-05 이동
│                             │
│       [다시 보정하기]         │  ← SCR-01 이동 (원문 재입력)
└─────────────────────────────┘
```

**구성 요소**
- 날짜 표시
- 입력 원문 카드 (전체 내용)
- 보정 결과 카드 (전체 내용 + 확률)
- 출처/근거 보기 버튼 (SCR-05 이동)
- 다시 보정하기 버튼 (SCR-01 원문 재입력)
