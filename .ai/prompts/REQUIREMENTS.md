# 요구 사항 명세서 (기획서, 기능 정의서)

## 프로젝트 개요

**앱 이름**: Cluein (클루인)
**태그라인**: 단서를 넣어(Clue-in) 기억을 완성하는 프리미엄 추론 서비스
**핵심 컨셉**: "그거 있잖아~" 하는 인간의 모호한 화법을 분석하여 기억을 보정해 주는 앱

### 포지셔닝
- 시장 위치: **검색과 상담 사이**
- 대상 문제: Tip-of-the-tongue(설단 현상) — 분명히 알고 있는데 검색어를 못 찾는 상태

---

## 핵심 사용 시나리오

### 시나리오 A: 파편화된 뉴스 기억 복원
- **상황**: 사용자가 과거에 본 뉴스를 찾고 싶으나 키워드가 불분명
- **예시**: "사막에서 캡슐 기차 타는 뉴스, 2005년이었나?"
- **흐름**:
  1. 사용자가 생각나는 대로 자유롭게 입력
  2. 불확실한 부분(연도, 이름 등)을 UI에서 표시
  3. 엔진이 키워드 추출 후 사실 여부 대조
  4. 보정 결과 제시: "2005년에는 기술이 없었습니다. 2016년 네바다 사막 테스트 뉴스일 확률이 95%입니다."
  5. 실제 뉴스 링크 + 관련 영상 제공

### 시나리오 B: 영화/콘텐츠 기억 복원
- **상황**: 영화 제목이 생각나지 않고 장면·설정만 기억
- **예시**: "태어날때부터 역할이 정해져있고 남녀들이 올백 머리를 하고 있고 2000년대 해외 영화"
- **흐름**:
  1. 사용자가 기억나는 특징을 서술
  2. 엔진이 특징 기반으로 후보 추론
  3. 정답 + 근거 제시: "가타카(Gattaca, 1997)일 가능성이 매우 높습니다. 이유: ..."

### 시나리오 C: 만델라 효과 / 잘못된 상식 보정
- **상황**: 내 기억이 맞는지 친구와 내기 또는 스스로 의심
- **흐름**:
  1. 사용자가 자신의 기억을 입력
  2. 실제 사실과 대조
  3. "사용자님은 A라고 하셨지만, 실제로는 B입니다. 그 이유는 C 때문인 것으로 보입니다." 형식으로 설명

## 기능 요구 사항

### FR-01. 자유 입력 인터페이스
- 사용자가 단편적인 기억을 자연어로 자유롭게 입력할 수 있다
- 입력 힌트(placeholder)로 모호한 표현을 유도한다

### FR-02. 확신도 표시 (흐릿함 마킹)
- 사용자가 입력 중 불확실한 부분을 선택하여 "긴가민가" 표시를 할 수 있다
- UI: 텍스트 드래그 선택 또는 태그 방식

### FR-03. 기억 보정 추론 엔진 (Gugeo Engine)
- 입력에서 핵심 키워드를 추출한다
- 확실한 정보 / 불확실한 정보를 분리하여 처리한다
- 사실 여부를 검증하고 보정 결과를 생성한다
- 추측인 경우 반드시 "추측" 또는 확률로 표현한다

### FR-04. 보정 결과 표시
- 원래 사용자 입력과 보정된 사실을 나란히 보여준다
- 보정 이유를 친절하게 설명한다
- 형식: "사용자님은 [A]라고 하셨지만, 실제로는 [B]입니다. [C] 때문입니다."

### FR-05. 출처 및 근거 제공
- 보정 결과에 실제 뉴스 링크, 영상, 참고 자료를 함께 제공한다
- 추측 정보는 출처 없이 확률로만 표현한다

### FR-06. 보정 이력 저장
- 사용자의 과거 보정 이력을 조회할 수 있다
- 각 이력에는 입력 원문, 보정 결과, 날짜가 포함된다

## 프로젝트 기술 요구 사항

### NFR-01. 플랫폼
- Android (minSdk 26, targetSdk 34)
- 언어: Kotlin, Jetpack Compose

### NFR-02. UI/UX
- 대화형(채팅형) 인터페이스
- 모호한 입력을 자연스럽게 유도하는 UX

### NFR-03. AI 통합
- Gugeo Engine은 **Google Gemini API** 를 사용한다
- 추측과 사실을 명확히 구분하는 응답 형식

### NFR-04. 보안
- API 키는 소스코드에 포함하지 않는다
- API 키는 DB에 저장하지 않는다
- API 키는 파일(local.properties 등)에 저장하지 않는다
- API 키는 **앱 실행 시 사용자가 직접 입력**하거나, 안전한 런타임 환경(예: 환경변수, Android Keystore, 서버 프록시)을 통해서만 공급한다

## 용어 정의

| 용어 | 설명 |
|---|---|
| Gugeo Engine | 내부 추론 엔진. "그거" + "추론"의 합성어. Google Gemini API 기반 |
| 기억 보정 | 사용자의 부정확한 기억을 실제 사실과 대조하여 수정·보완하는 과정 |
| 설단 현상 (Tip-of-the-tongue) | 알고 있지만 말로 표현이 안 되는 상태 |
| 만델라 효과 | 집단적으로 공유되는 잘못된 기억 |
| 확신도 | 사용자가 자신의 기억에 대해 느끼는 확실성의 정도 |
