# Cluein (클루인)

> 단서를 넣어(Clue-in) 기억을 완성하는 프리미엄 추론 서비스

"그거 있잖아~" 하는 모호한 화법을 분석하여 기억을 보정해 주는 Android 앱입니다.

---

## 포지셔닝

**검색과 상담 사이** — 분명히 알고 있는데 검색어를 못 찾는 상태(설단 현상)를 해결합니다.

| 상황 | 예시 |
|---|---|
| 뉴스 기억 복원 | "사막에서 캡슐 기차 타는 뉴스, 2005년이었나?" |
| 영화/콘텐츠 기억 복원 | "태어날때부터 역할이 정해져있고 올백 머리를 하고 있던 2000년대 해외 영화" |
| 만델라 효과 보정 | 내 기억이 맞는지 실제 사실과 대조 |

---

## 주요 기능

- **자유 입력**: 단편적인 기억을 자연어로 자유롭게 입력
- **확신도 마킹**: 불확실한 부분을 "긴가민가" 표시
- **기억 보정 (Gugeo Engine)**: Google Gemini API 기반 추론 — 사실과 추측을 명확히 구분하여 결과 제공
- **출처 제공**: 보정된 사실에 대한 뉴스 링크·영상·참고 자료 제공
- **보정 이력**: 과거 보정 기록 저장 및 조회

---

## 기술 스택

| 항목 | 내용 |
|---|---|
| 플랫폼 | Android (minSdk 26 / targetSdk 34) |
| 언어 | Kotlin |
| UI | Jetpack Compose |
| 추론 엔진 | Gugeo Engine (Google Gemini API) |

---

## 보안

- API 키는 소스코드, DB, 파일 어디에도 저장하지 않습니다
- 앱 실행 시 사용자가 직접 입력하며, Android Keystore를 통해 암호화하여 런타임에만 사용합니다

---

## 용어

| 용어 | 설명 |
|---|---|
| Gugeo Engine | "그거" + "추론"의 합성어. Google Gemini API 기반 내부 추론 엔진 |
| 기억 보정 | 부정확한 기억을 실제 사실과 대조하여 수정·보완하는 과정 |
| 설단 현상 | 알고 있지만 말로 표현이 안 되는 상태 (Tip-of-the-tongue) |
| 만델라 효과 | 집단적으로 공유되는 잘못된 기억 |
| 확신도 | 사용자가 자신의 기억에 느끼는 확실성의 정도 |
